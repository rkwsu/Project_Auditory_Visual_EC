Please download all folders to the C:\\ drive on your local machine.



Under the Code directory, place the following dependencies:



fsaverage



FieldTrip



These toolboxes are not included in this repository and must be downloaded separately by each user.



The file TE\_M\_movie\_PN\_AUD.m is the MATLAB script for generating movies for the Auditory Naming Task, and TE\_M\_movie\_PN\_VIS.m is the MATLAB script for the Visual Naming Task.



The file TE\_run.m is the MATLAB script for performing the Transfer Entropy (TE) calculation.



All TE-related scripts are located in Code/TE.



This analysis uses the sample data located in Positive\_AUD/TE1.
